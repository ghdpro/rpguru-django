Welcome to RPGuru

This site is about Japanese games (as the name implies, RPGs mostly) and whether or not they retained the Japanese voice track when they were localized and released in North America and Europe. There are some that may find it odd, but some people simply prefer playing games with the original voice track.

To be able to see which games properly support dual audio or not, look for the following icons next to the game titles:

  • Game has dual audio option
  • Possible issue, see description
  • Game doesn't have dual audio option
